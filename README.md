# Backend eMercado - Proyecto Final Grupo 6

Este es el backend de nuestro proyecto eMercado. Es una API que devuelve los datos necesarios para el funcionamiento de la página, realiza el login con JWT y guarda en una base de datos, los pedidos al finalizar la compra.

## Cómo ejecutarlo
En la terminal:
```bash
node server.js
```

El servidor va a arrancar en `http://localhost:3000`

## Para la Base de datos

Instalar extensión DBCODE, crear conexión.
  -MariaDb
  -Nombre: "ecommerce"
  -host: "localhost"
  -usuario: "root"
  -contraseña: 123
  -Almacenamiento de contrasela: "No almacenar"
  -Actualizar base de datos
  -Rol: Desarrollo
  -Guardar

Efecutar archivo /backend/ecommerce.sql en conexion creada poniendo contraseña.

Volver a la extencion DBCODE y abrir ecommerce - ecommerce - Tabla - pedidos: aqui se podran ver los pedidos que se realicen al finalizar la compra en el carrito.
